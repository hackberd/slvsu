Der Ordner Slivisu_Siedlungsdaten hat folgende Bestandteile 

.classpath, .project : Dateien für eclipse

Ordner:

src: Quellcode von Slivisu

jar: die zum Kompilieren notwendigen JARs

lib32/lib64: die zur Laufzeit notwendigen JOGL-Bibliotheken, 
die zur Darstellung der Karte benötigt werden.
lib32 für 32bit-Systeme und lib64 für 64bit-Systeme.

data: Speicherort für Daten, wird vom Programm automatisch geöffnet
